# Local COKIT guest SDK

This folder is a local stand-in for the official COKIT Tauri guest API (`@1io/tauri-plugin-co-sdk-api` or similar from [1iolabs/cokit](https://github.com/1iolabs/cokit)). That package is not on npm yet, so the app vendors/adapts the guest-js bindings here.

When the real package is published, replace this folder with a dependency on it.
